# CashFlow - 个人财务管理 Web 应用

## 项目概述

CashFlow 是一个基于 HTML + CSS + JavaScript 构建的现代化个人财务管理 Web 应用。应用采用响应式设计，支持桌面和移动设备，提供完整的记账、账单管理、统计分析和预算控制功能。

## 项目结构

```
src/
├── README.md              # 项目说明文档
├── css/
│   └── styles.css         # 全局样式文件
├── js/
│   ├── navbar.js         # 导航栏组件
│   └── utils.js          # 工具函数库
├── index.html            # 主页（仪表板）
├── login.html            # 登录页面
├── register.html         # 注册页面
├── add-transaction.html  # 记账页面
├── transactions.html     # 流水账单
├── analysis.html         # 统计分析
├── budget.html          # 预算管理
└── account.html          # 账户中心
```

## 功能特性

### 1. 用户系统
- 用户注册和登录
- 用户资料管理
- 账户安全设置

### 2. 核心功能
- **记账功能**：快速记录收入和支出
- **账单管理**：查看历史交易记录，支持筛选和搜索
- **统计分析**：收支趋势图、分类占比分析
- **预算管理**：设置月度预算，跟踪预算执行情况
- **账户中心**：个人资料管理、数据导出

### 3. 技术特点
- 响应式设计，完美适配移动端和桌面端
- 现代化的 UI 设计，使用 Tailwind CSS
- 本地存储数据，支持离线使用
- 模块化的 JavaScript 架构
- 统一的导航系统

## 使用指南

### 1. 运行应用

直接在浏览器中打开 `index.html` 文件即可运行应用。

### 2. 初始使用

1. 首次访问会自动跳转到登录页面
2. 点击"立即注册"创建新账户
3. 登录后即可开始使用各项功能

### 3. 主要功能说明

#### 记账页面
- 选择收入或支出类型
- 输入金额和选择分类
- 设置日期和添加备注
- 支持快速复制上次记录

#### 流水账单
- 按时间查看交易记录
- 支持按日期、分类筛选
- 搜索交易记录
- 导出交易数据为 CSV

#### 统计分析
- 月度收支趋势图表
- 支出分类占比分析
- 财务洞察和建议
- 数据可视化展示

#### 预算管理
- 为不同分类设置月度预算
- 实时跟踪预算执行情况
- 超支提醒功能
- 预算设置管理

#### 账户中心
- 个人资料编辑
- 密码修改
- 数据导出
- 账户安全设置

## 技术栈

- **前端框架**：纯 HTML5 + CSS3 + JavaScript
- **样式框架**：Tailwind CSS
- **图标库**：Material Symbols
- **图表库**：Chart.js（统计分析页面）
- **数据存储**：localStorage（浏览器本地存储）

## 开发说明

### 代码规范

1. **HTML 结构**：语义化标签，良好的可访问性
2. **CSS 样式**：使用 CSS 变量统一主题，响应式设计
3. **JavaScript**：模块化开发，避免全局污染

### 样式系统

应用使用 CSS 自定义属性（变量）实现主题系统：

```css
:root {
    --primary-color: #003c90;
    --income-emerald: #10B981;
    --expense-rose: #EF4444;
    /* 更多颜色变量 */
}
```

### 工具函数

`utils.js` 提供了常用的工具函数：

- `formatDate` - 日期格式化
- `formatCurrency` - 金额格式化
- `storage` - 数据持久化
- `statistics` - 统计计算
- `notification` - 通知提示
- `validation` - 表单验证

### 导航系统

`navbar.js` 管理底部导航栏：

- 自动根据当前页面激活对应导航项
- 统一的导航样式和交互
- 未登录用户自动跳转到登录页

## 浏览器兼容性

- Chrome 80+
- Safari 13+
- Firefox 75+
- Edge 80+

## 注意事项

1. 应用数据存储在浏览器本地，清除浏览器数据会丢失所有记录
2. 建议定期导出备份数据
3. 暂不支持云端同步功能（未来版本将添加）
